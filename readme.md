# Raylib Game

Simple boilerplate 

## PREREQUISTS

- `brew install raylib` or download raylib

- `cd build`
- `cmake ..`
- `make`
- `./Game`


## VS-Code

The `.vscode/c_cpp_properties.json` contains the include path of echo `pkg-config --libs --cflags raylib`
